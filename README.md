# DevOps
Tp Développement Opérationnel

Pour déployer la machine virtuelle et l'environnement de travail :<br/>
Au préalable, avoir téléchargé Vagrant et Ansible<br/>
Se mettre dans le dossier vagrant<br/>

`cd vagrant`<br/>
`vagrant up`

Déploiement de:<br/>
-Nginx <br/>
-Sqlite<br/>
-Supervisord<br/>
-gunicorn<br/>
-flask<br/>
